# Private Agent Suite Example

This is a sample private project configuration for Marten that demonstrates how to set up a private agent suite using only public surface APIs.

## Structure

- `agents.json` - Agent definitions for this private project
- `platform.json` - Platform configuration
- `models.json.example` - Template for model configuration (rename to use)
- `mcp.json.example` - Template for MCP server configuration (rename to use)
- `skills/` - Custom skills for this project
- `private_docs/` - Private documentation and knowledge base

## Usage

This project demonstrates:
- Configuring builtin agents without internal modules
- Setting up endpoint bindings for external integrations
- Defining private retrieval domains for knowledge access

See the main Marten documentation for how to use this configuration in your workspace.
