# Private Agent Suite Example

This is a sample private project that demonstrates using the Marten framework
with only public surface APIs.

## Structure

- `agents.json` - Agent definitions
- `platform.json` - Platform configuration
- `models.json.example` - Example model configuration (rename to use)
- `mcp.json.example` - Example MCP configuration (rename to use)
- `skills/` - Custom skills for this project
- `private_docs/` - Private documentation files

## Usage

Copy `models.json.example` to `models.json` and `mcp.json.example` to `mcp.json`,
then configure according to your needs.
