# Phase 7 Plan

> 阶段名称：玄学 Agent
> 目标：验证“规则步骤 + 知识引用 + 用户配置分析维度”型复杂回答工作流
> 对应设计：[phase-7-metaphysics-agent.md](/Users/litiezhu/workspace/github/youmeng-gateway/docs/architecture/phase-7-metaphysics-agent.md)

## 一、阶段目标

本阶段在前面平台、RAG 和知识型 agent 稳定之后，再引入规则更复杂、可解释性要求更高的玄学工作流。

完成后应具备：

1. 接收八字问题
2. 按用户配置的分析维度执行
3. 结合 RAG 和规则步骤组织回答
4. 输出步骤化、可解释结果

## 二、范围

### 本阶段要做

- 八字问题识别
- 维度配置加载
- 步骤化分析
- RAG / 文档引用整合

### 本阶段不做

- 完整命盘系统
- 自动长期用户画像
- 多流派并行分析

## 三、核心任务

### Task 7.1 分析维度配置

- 支持用户定义维度
- 支持顺序和权重

### Task 7.2 步骤化分析工作流

- 按固定步骤执行
- 记录每一步输出

### Task 7.3 文档引用与结果整合

- 从配置文档 / RAG 中取依据
- 生成最终解释

## 四、阶段产出

- Metaphysics Agent workflow
- 分析维度配置模型
- 步骤化解释模板

## 五、阶段通过标准

- [ ] 可加载用户配置的分析维度
- [ ] 可按步骤生成解释
- [ ] 可引用文档依据支撑回答
